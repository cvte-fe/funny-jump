# funny-jump
包教不包会，60分钟写个游戏小程序

### 它是什么？
这是一个非常有趣的项目，只要跟着教程一步一步的学习，就算是前端小白也能顺利完成一个微信小程序的游戏开发。

### 游戏内容
* 这个一个横版刷分游戏
* 通过手机语音识别，根据你喊出声音的大小去控制人物跳跃的高度，避开路上的障碍
* 可分享给好友（待定）

### 你能学到什么？
* 知道基本的微信小程序开发流程
* 了解游戏设计的基础知识
* 学会调用相关API（笑）
